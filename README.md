# Odin-RPS
This is an odin project designed to improve js
Odin gave the task to make a rock paper scissors 

The object of the game is first to 5 wins 


Pseudocode 

Have an input for the player to write either rock, paper, or scissors

Next have a system like an ai that chooses rock paper or scissors 

Go into a boolean response to determine the winner
If player chooses "Rock" or "Paper" or "Scissors"
  then can move on
else output not applicable 

If player chooses rock and ai chooses rock 
  output tie 
else if ai chooses paper 
  output ai wins 
else player wins 
If player chooses paper and ai chooses paper 
  output tie 
else if ai chooses scissors 
  output ai wins 
else player wins 
If player chooses scissors and ai chooses scissors 
  output tie 
else if ai chooses rock 
  output ai wins 
else player wins
